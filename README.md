# templated_matrix_class
This repository has the basic version of a templated matrix class in C++. This class is also compatible with complex numbers. For more information on this repository, please contact [bansalakash96@gmail.com](mailto:bansalakash96@gmail.com).

In the repository "templated_matrix_class", there is a header file named "matrix.h" which can be included in C++ programs to perform basic matrix operations such as matrix addition, matrix subtraction, matrix multiplication, scalar-matrix multiplication, scalar-matrix division, etc.
This matrix class is a templated class that uses dynamic memory in C++. One can make use of numerous datatypes such as 32-bit float, 32-bit int, 64-bit double, 64-bit long int, 128-bit long double (in an x86_64 architecture machine) to perform various matrix operations.

The elements in the matrix class are also compatible with the "Complex Numbers" data type (defined in "complex.h").
